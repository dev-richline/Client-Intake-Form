Richline — Client Intake Form (نموذج استلام عميل جديد)

This repository contains the official client intake form used by Richline to onboard new clients.

The form is designed to collect comprehensive details about the client's brand, marketing goals, target audience, and available assets in a secure and organized manner.

Contents

index.html — The complete, self-contained form interface (HTML + CSS + JS).

LICENSE — Strict Richline Proprietary License (Private Use Only).

robots.txt — Instructions for search engine crawlers.

sitemap.xml — Site map for SEO purposes.

Key Features

Secure Data Handling: Submissions are sent securely to Google Apps Script.

Anti-Spam Protection: Includes a "Honeypot" field to prevent bot submissions.

Safety Compliance: Does not request sensitive passwords directly (uses "Access/Notes" fields) to prevent browser security warnings.

Responsive Design: Fully adaptable layout using CSS Grid.

Deployment Instructions

Create a private GitHub repository (recommended due to the proprietary nature).

Upload all included files (index.html, LICENSE, robots.txt, sitemap.xml) to the repository root.

Enable GitHub Pages:

Go to Settings → Pages

Select Deploy from branch

Choose the main branch and click Save

Your form will be accessible at your GitHub Pages URL.
